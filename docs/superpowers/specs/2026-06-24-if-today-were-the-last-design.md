# 《如果只剩今天》/ If Today Were the Last — 設計

一個討論生命議題的互動自省作品，Harry 個人品牌的第一個數位互動藝術作品。
純前端、零後端，作為一個 Astro 頁面放進 harryfan.github.io 根站。

## 核心

- **形式**：選項式問答自省。8 題關於「為己 vs 為人」的抉擇。
- **折磨機制**：選了立刻定格，**不可回頭、不可修改**。畫面打出「已經過去」式提示，題目沉入背景。模擬人生不可逆。
- **進度隱喻**：用遞減的「剩餘天數」呈現（呼應命名與生命流逝），非進度條。
- **結尾**：個人化結語卡片 — 量化反映選擇（為己 M 次 / 為人 N 次）+ 一句依比例變化的收束。提供 `[分享]`（複製文字）與 `[回饋想法]`（mailto）。

## 視覺

暗色沉浸：黑底、米白字、大量留白、文字淡入。**刻意跳脫**主站紅黃 brutalist 風，建立獨立作品感。全螢幕，覆蓋 BaseHead 載入的 paper 風 body。

## 技術

- 單一 Astro 頁 `/today`（route = `src/pages/today.astro`）。
- Vanilla JS 狀態機：`intro → question[i] → ending`。零新依賴。
- 題庫為 JS 陣列；每個選項標記 `self` 或 `other`，結語前端計算。
- 選擇存記憶體陣列，不可返回（無上一頁按鈕、不存 localStorage 以保「不可逆」儀式感）。
- Header 加一個導覽連結。`BASE_URL` 為 `/`，路徑安全。

## 題庫主軸（8 題，二選一）

醒來第一念、想說的話、愧對的人、擱置的夢、陪伴與工作、原諒、最大的後悔、若今天是最後一天。每題一選項偏「為自己」、一選項偏「為別人/逃避」。

## 結語邏輯

依 self 次數分三段收束語（幾乎沒為自己 / 在兩者間反覆 / 選擇了自己），加精確次數。分享文字含站點連結；回饋走 `mailto:harry750110@gmail.com`。

## 不做（YAGNI）

無後端、無帳號、無資料收集、無排行榜、無多語。截圖分享先用複製文字，不做 canvas 出圖（之後可加）。
