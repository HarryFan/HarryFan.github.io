棚拍人像

以專業棚燈、柔光箱與乾淨背景，打造質感細膩、層次分明的商業人像。

風格介紹

棚拍人像（Studio Portrait）是商業攝影裡最講究「控制」的一門：在沒有自然光的棚內，攝影師完全靠燈具、修飾光與背景，從零建構出一張既真實又精緻的人像。它的目標是讓主體在乾淨環境中被最有質感地呈現——形象照、定妝照、廣告大片都屬於這個範疇。對 AI 下提示詞時，你其實是在「擺燈」：決定主光從哪打、補光補多少、背景留什麼，這些比人物長相更決定成敗。

技術核心是布光與鏡頭的搭配。鏡頭上，85mm portrait lens 搭 f/2.8 是棚拍黃金組合——焦段讓臉部透視自然不變形，光圈帶來柔和又不致糊掉細節的景深。布光則以 softbox key light with reflector fill 為基礎：柔光箱當主光製造柔順的明暗過渡，反光板補回暗部細節，控制光比就是控制情緒。進一步用 Rembrandt lighting（臉頰三角亮區）增加立體與個性，或換蝴蝶光走柔美路線。別忘了 soft catchlight 眼神光，它讓眼睛活過來；背景則用 clean grey seamless backdrop 把焦點全留給人。

題材與情緒上，棚拍人像適合需要專業、可信與精緻感的場景：企業形象照、演員定妝、美妝廣告、模特兒作品集。情緒可以從沉穩權威到柔美親和，全靠光比與背景色調拿捏。要拍得好，記得保留 detailed skin texture and pores，磨皮過頭會變蠟像；裁切與構圖也要留意肩線與頭頂空間，避免比例失衡。

各模型差異如下。Flux 對棚燈布光與皮膚質感的還原最自然，自然語言理解佳，直接描述主光、補光、背景就有商業片的乾淨感，是棚拍首選。Midjourney 質感漂亮、立體感強，但內建偏修圖味，想要中性真實時加 --style raw，比例用 --ar 4:5 貼近人像直幅。Stable Diffusion（SDXL 搭寫實人像 LoRA）可控性最高，CFG 抓 4～6、配 ADetailer 修臉、負面寫滿 harsh shadows, plastic skin，能逼出極細皮膚層次。GPT Image 強在對話迭代，可逐句要「主光移到左前」「補光多一點」「加點眼神光」「背景換淺灰」，對不懂布光的人最友善，但極致細節與一致性略遜前三者。我的流程常是：GPT Image 試布光與表情，再交 Flux 或 SD 出高質感終稿。

常見問題 FAQ

棚拍人像在哪個 AI 工具效果最好？

Flux 對皮膚質感與棚燈布光還原最自然，Midjourney 質感華麗，GPT Image 適合用對話微調光位與表情。

為什麼人像臉上有難看的死陰影？

主光太硬或缺補光。改用 softbox key light、加 reflector fill，必要時降低光比。

眼神光（catchlight）為什麼重要？

它讓眼睛反射出光源、顯得有神，缺了眼睛會呆滯，寫 soft catchlight 即可。

林布蘭光、蝴蝶光怎麼選？

林布蘭光（臉頰三角亮）顯立體有個性，蝴蝶光（鼻下蝴蝶影）顯柔美，依風格挑選。

背景該用什麼顏色？

商業形象照多用灰色或白色無縫背景，深色顯穩重、淺色顯乾淨，避免雜亂或高彩度搶戲。

怎麼避免皮膚被磨成塑膠？

寫 detailed skin texture and pores，並在負面加 plastic skin，保留毛孔比完美更可信。

中文還是英文 prompt 比較準？

布光與鏡頭術語英文最精準，建議以英文 softbox、Rembrandt 等詞為主，中文輔助描述人物與情緒。

怎麼讓同一個人多張形象照一致？

固定臉部描述、布光與 seed，Midjourney 用 cref，Flux 或 SD 可搭 LoRA 或 IP-Adapter 鎖人物。
