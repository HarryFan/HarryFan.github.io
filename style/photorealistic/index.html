超寫實攝影

風格介紹

超寫實攝影（Hyperrealism）的目標不是「漂亮」，而是「以假亂真」——讓觀者第一眼相信這是一張用真實相機拍下的照片。它的根源來自商業攝影與後期修圖工業：精準的曝光、可信的膚色、符合物理的光影衰減。當你對 AI 下這類提示詞時，本質上是在描述一整套攝影器材與現場條件，而不是套濾鏡。因此最關鍵的心法是「像個攝影師一樣思考」：先決定相機與鏡頭，再決定光，最後才談氛圍。

技術核心在於三件事如何協同運作。第一是鏡頭語彙：85mm prime lens 帶來人像最討喜的透視壓縮，f/1.8 大光圈製造奶油散景與主體分離，焦段與光圈一改，整張照片的空間感就完全不同。第二是光線：soft natural window light 之所以是寫實照片的靈魂，是因為它有方向、有衰減、有柔和的過渡，避免棚燈那種死板的塑膠味。第三是皮膚質感：真實皮膚會有毛孔、細紋與 subsurface scattering（光穿入皮膚再透出），這是區分照片與 3D 渲染的分水嶺，磨皮過頭立刻變蠟像。

題材上，超寫實最適合人像寫真、形象照、商品廣告與時尚視覺，凡是需要「讓人相信這是真的」的場景都適用；但它不擅長奇幻、誇張或情緒化的戲劇張力，那是電影感與概念藝術的舞台。情緒上它偏向中性、克制、可信，靠細節而非戲劇取勝。實拍時記得把對焦錨點放在眼睛，保留適度瑕疵，反而比完美無瑕更有說服力。

各模型實戰差異明顯。Flux 在開源陣營對皮膚物理與自然光的還原最強，長句自然語言理解佳，直接寫滿相機、鏡頭、光源就有照片感，是目前超寫實首選。Midjourney 質感華麗、細節豐富，但內建調色偏向「雜誌修圖」，想要中性真實時要加 --style raw 並降低風格化，比例用 --ar 3:2 貼近相機。Stable Diffusion（SDXL 搭寫實 LoRA 如 RealVis）可控性最高，CFG 抓 4～6、配上 ADetailer 修臉與手、負面寫滿 cartoon, illustration, plastic skin，能逼出極高細節。GPT Image 勝在對話迭代，可以一句句要「光再側一點」「皮膚別磨太兇」「景深更淺」，對不懂參數的人最友善，但極致細節與一致性仍略遜前三者。我的流程通常是：Flux 或 SD 出高細節終稿，GPT Image 做快速構圖試驗。

常見錯誤

  • 皮膚過度磨皮失去毛孔顯塑膠感
  • 缺少明確鏡頭與光圈導致景深平淡
  • 光源方向矛盾產生不自然陰影
  • 飽和度過高失去真實膚色
  • 眼睛失焦讓整張照片失去說服力

常見問題 FAQ

超寫實攝影在哪個 AI 工具效果最好？

Flux 對皮膚毛孔與光線物理的還原最自然，Midjourney 質感華麗但偶爾偏修圖味，GPT Image 適合用對話微調光線與構圖。

為什麼生成的人臉看起來像塑膠或蠟像？

多半是磨皮過頭。把 photorealistic skin pores、subsurface scattering 寫進正向，並在負面加入 plastic skin、airbrushed。

怎麼避免一看就知道是 AI 圖？

指定具體相機、鏡頭、光圈與光源方向，保留瑕疵與毛孔，避免過高飽和度與完美對稱。

該選哪顆鏡頭焦段？

人像用 85mm 或 50mm，環境帶景用 35mm，商品特寫用 100mm 微距，焦段會直接影響透視與景深。

超寫實和電影感差在哪？

超寫實追求中性真實的色彩與光線，電影感則刻意加入調色、寬畫幅與戲劇打光，兩者可疊加但目標不同。

為什麼手指常常畫錯？

這是模型通病，建議固定 seed 多生幾張挑選，並把 deformed hands、extra fingers 放進負面提示。

中文還是英文 prompt 比較準？

攝影術語英文最精準，建議以英文鏡頭光圈詞為主，中文僅輔助描述情緒與場景。

如何維持同一個人多張照片一致？

固定臉部描述與 seed，Midjourney 用 cref，Flux 與 SD 可搭配 LoRA 或 IP-Adapter 鎖定人物。
